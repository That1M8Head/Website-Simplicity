Website simplicity

Why is this website so simplistic?

Look, if I wanted to decorate this website with a bunch of images darted around the place, I sure as hell wouldn't be doing it in bare-bones HTML. And that's fine by me.

The time it takes

I'm sure I could absolutely redo this website and make it fancy and 'professional' if I was willing to put in the time. Thing is, I'm not, and I don't need to be. A website doesn't need to have a billion images on it with ads and cookies and tracking.

If I wanted a website like that, I wouldn't make it myself.

The original WWW

The original World Wide Web, as proposed by Tim Berners-Lee, was intended as a 'universal linked information system'. You can learn more about that by watching this video.

Businesses and companies started making their own websites. I'm not going to go into the history of that, check this Wikipedia page if you want to know.

Nowadays, you can't visit any major website without being bombarded by cookie popups and advertisements. Not to mention tracking.

The problem of 'website builders'.

You must have heard of website builders before. You know, Wix, Squarespace, all that? They act like the only way you can build a 'good looking website' is with one of those tools.

Well, guess what?

This web page, this one you're on right now, is written in pure HTML and CSS. No help from website builders. And it looks amazing.

JavaScript

Now, don't get me wrong, I love JavaScript. Can't get enough of it. But it doesn't need to be there all the time.

Most websites you'll see will almost always have JavaScript. Mostly just for ads, tracking and analytics. Even for websites that don't seem to need it.

Not all websites should be like this

My point wasn't "you MUST have a simplistic website or I will come to your house with a rocket launcher". I'm saying that not all websites need fancy design to be a good website.

Especially for one like this.

If you have an e-commerce thingy going on, then sure, you can go ahead and use a website builder. Again, I'm not saying don't use external tools. I'm saying that you don't need external tools for a good website.

Don't overcomplicate and break what already works perfectly.

Like how this website looks?

You can go ahead and download the simple.css from the GitHub repo! (link below) It's a tiny CSS document, literally less than a kilobyte. It even includes dark mode, y'know, if you swing that way.

All you need to do after copying it to your website's root directory is include this in your <head> block:

<link rel='stylesheet' type='text/css' href='simple.css'>

Once that's done, your website will now be formatted to look like this one. And this is free software, too, so you can go ahead and edit it if you want! Change the colours, spacing, add stuff, remove stuff, whatever!
